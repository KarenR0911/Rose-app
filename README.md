# Welcome to Rose App (Rose Robot Orientational System Educative)

Rose (Robot Orientational System Educative) es un robot asistente educativo interactivo diseñado para la orientación, acompañamiento y enseñanza de usuarios infantiles mediante inteligencia artificial conversacional, procesamiento de lenguaje natural y sistemas embebidos de interacción multimedia.
El sistema combina:
•	inteligencia artificial en la nube, 
•	procesamiento embebido, 
•	comunicación inalámbrica, 
•	reproducción de audio, 
•	animación visual, 
•	y una aplicación móvil de configuración y control. 
Rose está concebida como una plataforma educativa portátil capaz de interactuar de forma natural con el usuario mediante conversaciones, respuestas auditivas y expresiones visuales dinámicas.
